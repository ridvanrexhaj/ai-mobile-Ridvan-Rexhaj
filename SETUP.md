# Quick Setup Guide

## For Replit Users

1. Go to Replit Secrets (lock icon on left) and add:
   ```
   EXPO_PUBLIC_SUPABASE_URL=your_url_here
   EXPO_PUBLIC_SUPABASE_ANON_KEY=your_key_here
   ```
2. App starts automatically - done!

## For Expo Snacks Users

1. Go to [snack.expo.dev](https://snack.expo.dev)
2. Click "Import from GitHub"
3. Paste GitHub repo URL
4. Click "Env Variables" tab and add:
   ```
   EXPO_PUBLIC_SUPABASE_URL=your_url_here
   EXPO_PUBLIC_SUPABASE_ANON_KEY=your_key_here
   ```
5. Scan QR code with Expo Go to test on mobile
